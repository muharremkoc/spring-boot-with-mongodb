# spring-boot-with-mongodb
 This is Spring Boot MongoDB CRUD API


## Prerequisites

* Docker

* MongoDB

## Used Technologies

* Spring Boot 2.5.3
* MongoDB
* Spring Boot Web
* Docker



## Usage

- docker-compose up -d
- Please write application.properties for MongoDB
- Open Swagger or Postman
- Write CRUD operations
- Connect Database in Your IDE
- Controlled values
